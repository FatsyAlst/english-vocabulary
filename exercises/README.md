# Vocabulary Practice Exercises

This folder contains LaTeX-based practice exercises for the vocabulary words documented in this repository.

## Structure

Each practice set is organized in its own folder:

```
exercises/
├── README.md
└── vocabulary-practice-1/
    ├── vocabulary-exercises.tex    # Exercise document
    ├── vocabulary-exercises.pdf    # Compiled exercises
    ├── vocabulary-answers.tex      # Answer key document
    └── vocabulary-answers.pdf      # Compiled answer key
```

## Compiling the Documents

To compile the LaTeX files into PDFs, you need a LaTeX distribution installed (e.g., TeX Live, MiKTeX).

```bash
cd exercises/vocabulary-practice-1
pdflatex vocabulary-exercises.tex
pdflatex vocabulary-answers.tex
```
